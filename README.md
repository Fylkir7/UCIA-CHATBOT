# 🥗 UCIA Chatbot - Assistente Virtual de Nutrição

O **UCIA Chatbot** é um assistente virtual de nutrição desenvolvido para ajudar usuários com dúvidas alimentares, sugestões de dieta, e orientações básicas sobre hábitos saudáveis.

## 🚀 Funcionalidades

- Responde dúvidas sobre alimentação e nutrição
- Sugere cardápios equilibrados conforme objetivos (emagrecimento, ganho de massa, manutenção)
- Dá dicas de hábitos saudáveis e ingestão de água
- Suporte básico a intolerâncias (ex: lactose, glúten)

## 🧠 Tecnologias Utilizadas

- Python 3
- NLTK / spaCy (ou outro motor NLP, conforme seu código)
- Flask (para API ou interface web)
- JSON / SQLite (para armazenamento de dados, se aplicável)
